# **App Name**: Al-Salam ERP

## Core Features:

- Property Management: Manage property details including location, tenants, lease agreements, and maintenance schedules.
- Vehicle Management: Track vehicle information such as registration, maintenance, usage logs, and associated costs.
- Employee Management: Maintain records of employees and workers including personal information, job roles, and accommodations.
- Financial Management: Oversee financial operations including accounting, budgeting, and reporting. Includes accounts and billing
- Zakat Reporting: Generate reports compliant with the Zakat Authority of Saudi Arabia, using the data entered in the Financial Management features. Includes reports for owned real estate, and fleet.
- User Role Management: Define user roles and permissions to control access to different modules and functionalities.
- Accommodation Management: Manage the details of accommodation provided for employees and workers, including assigning, tracking occupancy, and costs.

## Style Guidelines:

- Primary color: Forest green (#228B22) to reflect growth, stability, and nature.
- Background color: Light green (#F0FFF0) to provide a calm and clean backdrop.
- Accent color: Darker green (#32CD32) to highlight important elements and actions.
- Body and headline font: 'Inter', a sans-serif font, to provide a modern and readable experience. Note: currently only Google Fonts are supported.
- Use consistent, professional icons across the application for easy navigation.
- Design a clean and structured layout with intuitive navigation.
- Incorporate subtle transitions and animations to enhance user engagement.
